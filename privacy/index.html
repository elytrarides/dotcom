Privacy Policy for Elytra LLC

1. Introduction

This Privacy Policy explains how Elytra LLC collects, uses, stores, and shares user data. By using our services, you agree to the collection and use of information in accordance with this policy.

2. Data Collection

We collect the following types of personal data:

  • Name
  • Phone Number
  • Profile Image
  • Location Data

Note: Payment processing for businesses is conducted by Stripe, and we do not store payment details on our servers.

3. Data Usage

User data is used to:

  • Facilitate rides by providing necessary details like pickup/dropoff locations, name, profile image, and passenger count.
  • Communicate with users regarding their reservations and ride status.
  • Enhance our service through data analytics.

4. Data Sharing and Disclosure

- We share phone numbers with Twilio for authentication and SMS updates regarding driver status.

- Driver locations are shared with riders in real time for service facilitation.

- User data may be disclosed if required by law.

- Text SMS opt-in data is not shared with 3rd parties.

5. Data Storage and Security

- User data is stored on our secured servers, accessible only by authorized Elytra LLC administrators.

- We implement robust security measures to protect user data.

6. User Rights and Choices

- Users can request the deletion of their reservations by contacting Elytra LLC.

- Users are informed about data collection through our use of cookies and third-party services like Google Maps API and Google Analytics.
